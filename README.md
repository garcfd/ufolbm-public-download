This code was developed from the 2D python code of Jonas Latt (unige coursera). The code here compiles with nvidia pgf90 fortran openacc gpu solver, although you can also compile it with standard gfortran. Lots of work has gone into the development of this LBM solver, mainly converting it to 3D, Fortran and making it run on nvidia GPU. The compiler generates executable versions. (host/multi/gpu). The solver imports STL files, and so can use arbitrary 3d geometry. The code uses RLBM regularised LBM method, work has also been used from Sangtani thesis (p55) and from Moritz Lehmann pdf notes on linkedin. You will obviously need a NVIDIA GPU if you want to run the GPU version, but the other 2 versions (basic and multicore) will run fine on just CPU. Recent work has focussed on developing a reflection matrix which can be used to model free-slip walls. Have a look on Youtube and Twitter/X for details of garcfd LBM work where you will find many examples and demos.
